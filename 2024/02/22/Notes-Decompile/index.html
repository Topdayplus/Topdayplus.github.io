标志位

CF进位标志

当运算结果的最高有效位有进位（加法）或借位（减法）时设置。

进位标志置1，即CF = 1；否则CF = 0

用途：用于表示两个无符号数高低。

举例：

3AH + 7CH＝B6H，没有进位：CF = 0

AAH + 7CH＝（1）26H，有进位：CF = 1

ZF零标志

运算结果为0则ZF=1，否则ZF=0。表示两个无符号数高低。
